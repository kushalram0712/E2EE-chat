# E2EE Chat Application

A secure, real-time messaging application built with **End-to-End Encryption (E2EE)**. This project ensures that messages are encrypted on the sender's device and can only be decrypted by the intended recipient, keeping data completely private from servers and third parties.

---

## ✨ Features

* **End-to-End Encryption:** Uses robust cryptographic algorithms (e.g., AES-GCM, RSA, or Diffie-Hellman) to secure messages.
* **Real-Time Messaging:** Instant message delivery powered by WebSockets.
* **Zero-Knowledge Architecture:** The server acts strictly as a relay and never has access to the plaintext messages or private keys.
* **Secure Key Exchange:** Safe cryptographic key generation and exchange protocol handled entirely client-side.
